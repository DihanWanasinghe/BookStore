📚 Welcome to the BookStore API

This is a RESTful API for managing books, authors, customers, carts, and orders.

BookResource (/bookstore/api/books)

  • POST /bookstore/api/books
  • GET /bookstore/api/books
  • GET /bookstore/api/books/{id}
  • PUT /bookstore/api/books/{id}
  • DELETE /bookstore/api/books/{id}

AuthorResource (/bookstore/api/authors)

  • POST /bookstore/api/authors
  • GET /bookstore/api/authors
  • GET /bookstore/api/authors/{id}
  • PUT /bookstore/api/authors/{id}
  • DELETE /bookstore/api/authors/{id}
  • GET /bookstore/api/authors/{id}/books

CustomerResource (/bookstore/api/customers)

  • POST /bookstore/api/customers
  • GET /bookstore/api/customers
  • GET /bookstore/api/customers/{id}
  • PUT /bookstore/api/customers/{id}
  • DELETE /bookstore/api/customers/{id}

CartResource (/bookstore/api/customers/{customerId}/cart)

  • POST /bookstore/api/customers/{customerId}/cart/items
  • GET /bookstore/api/customers/{customerId}/cart
  • PUT /bookstore/api/customers/{customerId}/cart/items/{bookId}
  • DELETE /bookstore/api/customers/{customerId}/cart/items/{bookId}

OrderResource (/bookstore/api/customers/{customerId}/orders)

  • POST /bookstore/api/customers/{customerId}/orders
  • GET /bookstore/api/customers/{customerId}/orders
  • GET /bookstore/api/customers/{customerId}/orders/{orderId}
